## Nvim plugin's Shortcut


### [kevinhwang91/nvim-hl](http://www.github.com/kevinhwang91/nvim-hlslens)

| Shortcut | Action                                        |
|----------|-----------------------------------------------|
| `*`      | Search for the next word under the cursor     |
| `#`        | Search for the previous word under the cursor |

|   Shortcut  |          Action          |
